AI-Powered Mental Health Monitoring System Technical Documentation

🔬 A. System Overview

The AI Mental Health Monitoring System is an intelligent platform designed to assess and monitor mental health status through a combination of biometric sensor data and psychological questionnaires. The system utilizes a high-accuracy ensemble machine learning model (90.28% accuracy) to classify stress levels into four distinct categories: Low, Moderate-Low, Moderate-High, and High.

By analyzing physiological signals such as heart rate, heart rate variability, respiration, skin temperature, and blood pressure, combined with psychological responses, the system provides real-time mental health insights and personalized recommendations.

✅ E. Model Selection Justification

🏆 Why Ensemble Learning?

Ensemble learning combines multiple weak or moderate learners to create a strong, robust model. By voting, different algorithms complement each other's weaknesses, improving generalization and reducing overfitting.

🎯 Algorithm Selection

Each algorithm was selected for specific strengths:

  • Extra Trees: Handles non-linear patterns, reduces variance
  • Gradient Boosting: Strong sequential learning, excellent for complex patterns
  • Random Forest: Robust feature importance, parallelizable
  • Logistic Regression: Interpretable baseline and probability calibration
  • K-NN: Local pattern detection, distance-based insights

📈 Performance Advantage

The voting ensemble achieves 90.28% accuracy, which exceeds the best individual algorithm (Extra Trees: 89.2%). This 1% improvement translates to fewer misclassifications and better clinical reliability.

🛡️ Robustness & Generalization

Cross-validation results (85.00% ± 3.16%) demonstrate the model's stability across different data splits. The low standard deviation indicates consistent performance and reduced risk of overfitting.

🏗️ F. System Architecture

Input Features (16 Total)

The model processes 16 engineered features derived from 8 base physiological inputs and psychological assessments:

🫀 Biometric Features (8)

  • Heart Rate (BPM)
  • Heart Rate Variability (HRV)
  • Respiration Rate
  • Skin Temperature
  • BP Systolic
  • BP Diastolic
  • Cognitive Assessment Score
  • Emotional Assessment Score

🧮 Engineered Features (8)

  • HR/HRV Ratio
  • Mean Arterial Pressure (MAP)
  • Pulse Pressure
  • Psychological Score (Cognitive + Emotional)
  • HR/Respiration Ratio
  • Temperature Deviation
  • Normalized HRV
  • HR Variability Index

Output Classification

The model outputs one of four stress level categories:

😌 Low

Baseline, healthy stress level

😐 Moderate-Low

Mild stress detected

😟 Moderate-High

Significant stress present

😰 High

Severe stress warning

The Receiver Operating Characteristic (ROC) curve demonstrates the model's ability to distinguish between different stress levels. The Area Under the Curve (AUC) provides a comprehensive measure of classification performance across all threshold values.

ROC AUC Performance:

  • High Stress Detection: AUC = 0.98 (Near-perfect classification)
  • Moderate-High Classification: AUC = 0.94 (Strong performance)

  • Clinical Significance: AUC > 0.90 indicates excellent diagnostic capability
📊 Interpretation: The ROC AUC curve shows the model's excellent ability to discriminate between stress levels. An AUC of 0.95 means the model correctly ranks 95% of randomly selected positive-negative pairs, demonstrating superior diagnostic performance for clinical applications.
